# 📊 SalesProfit-Analyzer: SQL & BI Insights

Este projeto demonstra a aplicação de **SQL Avançado** para resolver problemas reais de negócio, focando em análise de faturamento e rentabilidade.

## 🛠️ Tecnologias Utilizadas
- **Banco de Dados:** PostgreSQL
- **Ferramenta de Gestão:** pgAdmin 4
- **Linguagem:** SQL (DDL, DML e DQL)

## 💡 O que este projeto resolve?
Em um cenário de consultoria, é vital identificar não apenas o que vende mais, mas o que traz mais **lucro real**. Este script realiza:
1. **Modelagem Relacional:** Criação de tabelas com chaves estrangeiras (`Foreign Keys`) para garantir integridade.
2. **Cálculos de BI:** Uso de funções agregadas (`SUM`, `GROUP BY`) para calcular Margem de Lucro Percentual e Lucro Líquido por produto.

## 🚀 Como visualizar o resultado
O script principal gera uma visão executiva com as seguintes métricas:
- Total de unidades vendidas
- Faturamento Bruto
- Custo Total
- **Margem de Lucro %**
